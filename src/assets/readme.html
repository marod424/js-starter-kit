JS Starter Kit

Intro

JavaScript development environment starter kit.

Features

  • EditorConfig for cross editor JS coding style consistency
  • Package management via yarn/npm
  • Development Web Server - Express
  • npm scripts
  • Bundling with Webpack
  • Linting via ESLint
  • Mock data via JSON Server

Install

  1. Download node/npm
  2. Install yarn
  3. Clone the repo git clone [repo_url]
  4. Install dependencies cd js-starter-kit && yarn

Scripts

  • Run development server npm start
  • Build for production npm run build
  • Run tests npm test
